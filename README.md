# Python Learning & Experiments 

##  Proyectos y Scripts

Aquí encontrarás los experimentos desarrollados hasta el momento:

| Archivo | Descripción / Enfoque de Aprendizaje |
| :--- | :--- |
| **Bolas random.py** | Un script visual que genera 100 bolas de colores aleatorios y las posiciona de forma random en la pantalla (ideal para practicar con lógica aleatoria y entornos gráficos/dibujo). |
| **Contador manual.py** | Una pequeña herramienta o interfaz para llevar un registro o conteo manual, explorando el manejo de eventos y estados. |
| **Juego numero 1 al 10.py** | Un minijuego interactivo enfocado en la adivinación de números, perfecto para estructurar condiciones y bucles de juego de forma entretenida. |

---

## Objetivo del Repositorio

El código de estos archivos es intencionalmente sencillo. El foco principal es **aprender haciendo**: probar librerías dinámicas, entender cómo se comporta la aleatoriedad en Python y crear proyectos interactivos que vayan más allá de los textos planos en la terminal.
